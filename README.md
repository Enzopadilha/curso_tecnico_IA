# 🤖 Notebooks de Inteligência Artificial
Este repositório reúne notebooks desenvolvidos durante meu curso técnico em Inteligência Artificial. Aqui você encontrará conteúdos práticos, experimentos e anotações sobre os principais conceitos estudados ao longo das aulas.

## 📚 Objetivo
O objetivo deste repositório é:
- Registrar meu aprendizado em IA
- Organizar conteúdos estudados em aula
- Servir como material de revisão
- Compartilhar conhecimento com outras pessoas interessadas na área

## 🧠 Conteúdos Abordados

Os notebooks deste repositório podem incluir temas como:
- Introdução à Inteligência Artificial
- Lógica de Programação aplicada à IA
- Python para Ciência de Dados
- Manipulação de dados com Pandas
- Visualização de dados
- Machine Learning (Aprendizado de Máquina)
- Modelos supervisionados e não supervisionados
- Redes Neurais Artificiais
- Deep Learning (Aprendizado Profundo)
- Avaliação de modelos

## 🚀 Tecnologias Utilizadas
- Python
- Colab
- Bibliotecas:
- NumPy
- Pandas
- Matplotlib
- Statistics
